# 🏥 Healthcare Predictive Analytics Platform

An AI-powered Healthcare Predictive Analytics Platform built with **Streamlit**, **Machine Learning**, and **SHAP Explainable AI** to support clinical decision-making through transparent risk prediction and interactive healthcare analytics.

---

## ✨ Features

- 📊 Interactive Healthcare Analytics Dashboard
- 🤖 Machine Learning-based Patient Risk Prediction
- 🧠 SHAP Explainable AI for model interpretability
- 🩺 Patient-specific clinical recommendations
- 📈 Interactive visualizations using Plotly and Matplotlib
- 📂 Support for multiple healthcare dataset formats
- 🔍 Automatic dataset detection and preprocessing
- 🎨 Modern Streamlit user interface

---

## 🏗️ Supported Healthcare Datasets

The application supports multiple healthcare data formats:

- **MIMIC-III Style** – ICU patient vital signs and outcomes
- **Synthea Healthcare** – Synthetic electronic health records
- **NHANES Population Survey** – Population health analysis
- **Custom CSV Upload** – User-provided healthcare datasets

---

## 🧠 Machine Learning & Explainable AI

The platform uses:

- Gradient Boosting Classifier
- SHAP (SHapley Additive exPlanations)
- StandardScaler
- Patient-specific risk prediction
- Feature importance visualization
- Clinical recommendation generation

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/Healthcare-Predictive-Analytics.git
```

Navigate to the project directory:

```bash
cd Healthcare-Predictive-Analytics
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run enhanced_healthcare_app.py
```
